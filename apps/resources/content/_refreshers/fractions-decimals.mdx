A fraction **is a division**: 3/4 means 3 ÷ 4, which equals 0.75.

Worth knowing by heart:

| Fraction | Decimal |
| -------- | ------- |
| 1/2      | 0.5     |
| 1/4      | 0.25    |
| 3/4      | 0.75    |
| 1/10     | 0.1     |

Both forms name the same amount — 3/4 of a dollar and $0.75 are the same
money.
