An equation stays true as long as you do **the same thing to both sides**. To
find the unknown, undo what was done to it — in reverse order.

For example, solve 2x + 3 = 11:

1. Undo the "+ 3": subtract 3 from both sides → 2x = 8.
2. Undo the "× 2": divide both sides by 2 → x = 4.

Check by substituting back: 2 × 4 + 3 = 11. ✓
