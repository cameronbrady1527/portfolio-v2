A core Algebra I skill: deciding whether data is **linear** or **exponential**.
A linear function changes by a constant *amount* each step (you add); an
exponential function changes by a constant *ratio* (you multiply). These are
**real, released Regents questions** — read each table or description and pick
the model that fits.
