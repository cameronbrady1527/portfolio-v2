Here's the one rule that makes fraction addition click: **you can only combine
pieces that are the same size.** Quarters add to quarters, thirds to thirds — but
a third and a quarter have to be re-sliced to a common size first. Once the
denominators match, you just combine the tops. This card builds that habit on the
[equivalent fractions](/foundations/geometry/equivalent-fractions) you already
practiced.

<SkillCard skill="add-subtract-fractions" title="Adding & Subtracting Fractions">

<SkillStep title="What & why">

A denominator tells you the **size of each piece**: in $\frac{1}{4}$ the whole is
cut into four, so each piece is a "fourth." You can add $\frac{1}{4} + \frac{2}{4}$
directly — same-size pieces — to get $\frac{3}{4}$.

But $\frac{1}{2} + \frac{1}{4}$ mixes a half with a quarter: different-size
pieces, so the tops can't just be added. The fix is to rewrite one (or both)
fractions as **equivalent fractions with a common denominator**, so every piece
is the same size again. Then add the tops and keep the shared bottom.

</SkillStep>

<SkillStep title="Watch how it works">

**Same denominator? Add the tops.** Take $\frac{1}{5} + \frac{2}{5}$ — both are
fifths, so combine $1 + 2 = 3$ fifths:

$$\frac{1}{5} + \frac{2}{5} = \frac{3}{5}$$

**Different denominators? Re-slice first.** For $\frac{1}{2} + \frac{1}{4}$,
rewrite $\frac{1}{2}$ as $\frac{2}{4}$ so both are quarters. The bars below show
$\frac{1}{2}$ and $\frac{2}{4}$ covering the same amount:

<FractionBar
  fractions={[{ num: 1, den: 2 }, { num: 2, den: 4 }]}
  caption="1/2 = 2/4 — the same amount, now measured in quarters."
/>

Now both are quarters, so add the tops:

$$\frac{1}{2} + \frac{1}{4} = \frac{2}{4} + \frac{1}{4} = \frac{3}{4}$$

Subtraction works exactly the same way — match the denominators, then subtract the
tops. Always simplify the result at the end.

</SkillStep>

<SkillStep title="Reason through one together">

Find the common denominator first, then combine. Predict before you check.

<PredictThenCheck
  prompt="2/3 + 1/6 = ?"
  choices={["5/6", "3/9", "3/6", "1/2"]}
>

It's **5/6**. The denominators are $3$ and $6$; since $6$ is a multiple of $3$,
rewrite $\frac{2}{3}$ as $\frac{4}{6}$. Now both are sixths:
$\frac{4}{6} + \frac{1}{6} = \frac{5}{6}$. Adding the bottoms too (the $3 + 9$
trap) would be wrong — denominators name the piece size, they don't get added.

</PredictThenCheck>

</SkillStep>
